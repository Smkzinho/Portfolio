SOBRE MIM

Moro no Brasil, em uma pequena cidade chamada Boa Vista localizada na Paraíba. Sou apaixonado por tecnologia desde os 10 anos de idade.

Minhas habilidades

O Adobe Photoshop é um dos melhores softwares de edição de imagem da atualidade. Usado atualmente por muitas empresas de diversos ramos, e também por mídias socias.

O Figma é uma plataforma colaborativa para construção de design de interfaces e protótipos.

O Visual Studio Code (VS Code) é um editor de código de código aberto, o qual possui as funcionalidades mais simples como: edição de código com suporte a várias linguagens de programação, terminal de comandos integrado.

O illustrator tem sua função principal de trabalhar com vetores, todo tipo de projeto que for relacionado a esse formato pode ser feito no Illustrator. Os mais comuns são: Logos, Ícones, Fontes tipográficas.

O CapCut é uma plataforma de criação completa desenvolvida por IA que permite a edição de vídeos e criação de imagens em navegadores.

Essas são as três principais linguagens utilizadas no front-end da nossa aplicação, ou seja, são utilizadas do lado do cliente no próprio navegador.
